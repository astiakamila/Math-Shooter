# 🎮 Math Shooter

Math Shooter is an educational arcade game developed using **Python** and **Pygame**. Players solve randomly generated math problems by catching the correct falling answer while avoiding incorrect ones.

This project was originally developed as a university course assignment and later improved as part of our programming portfolio.

## ✨ Features

- Random math problem generation (+, −, ×, ÷)
- Three difficulty levels
- Character selection
- Player name input
- Score and lives system
- Dynamic falling speed
- Animated backgrounds
- Background music and sound effects

## 🛠 Technologies

- Python
- Pygame

## ▶️ How to Run

1. Install Pygame

```bash
pip install pygame
```

2. Run the game

```bash
python "MATH SHOOTER.py"
```
